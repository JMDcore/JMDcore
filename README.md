## Hola, soy JMDcore

Construyo herramientas digitales con foco en claridad, automatización y utilidad real. Me interesan especialmente las interfaces que ayudan a vender mejor, organizar procesos, visualizar datos y convertir ideas en productos que se puedan usar desde el primer día.

```ts
const jmdcore = {
  focus: ["web apps", "product systems", "automation", "clean UX"],

  stack: ["TypeScript", "JavaScript", "HTML", "CSS", "Docker"],
  mindset: "ship useful things, keep the interface calm, make the workflow obvious",
};
```

## Ahora mismo

- Trabajando en productos web útiles para freelancers, pequeños negocios y flujos comerciales.
- Mejorando proyectos con TypeScript, CSS moderno y despliegues cada vez más cuidados.
- Explorando automatización, agentes de IA y herramientas internas que ahorran tiempo.
- Abierto a colaborar en ideas con una mezcla de producto, datos e interfaz.

## Forma de trabajar

- Producto antes que decoración: cada pantalla tiene que resolver una acción concreta.
- Interfaces limpias: menos ruido, más foco, mejores decisiones.
- Automatización con criterio: ahorrar tiempo sin perder control.
- Iteración rápida: versión usable primero, mejora constante después.
